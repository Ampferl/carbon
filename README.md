# Carbon
Object oriented programming language in C. Soon it will be extendable and embeddable.

## Get Started
Build Carbon Interpreter:
```shell script
$ cd Carbon
$ chmod +x ./build.sh
$ ./build.sh
```
Run Carbon File:
```shell script
$ ./carbon [PATH-TO-FILE]
```
